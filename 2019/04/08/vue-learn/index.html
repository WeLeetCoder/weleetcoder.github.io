vue 学习

vue 模板语法

<div id="app">
    {{ message }}
</div>

var app = new Vue({
    el: '#app',
    data: {
        message: 'Hello Vue!'
    }
})

输出如下：

<div id="app">
    Hello Vue!
</div>

以上是最简单的 vue 的 hello world！

vue 条件、循环

判断
<div id="app-3">
    <p v-if="seen">现在你看到我了</p>
</div>

var app3 = new Vue({
  el: '#app-3',
  data: {
    seen: true
  }
})

通过改变 seen 的值，能够使得 p 标签显示和隐藏。

循环
<div id="app-4">
    <ol>
        <li v-for="todo in todos">{{ todo }}</li>
    </ol>
</div>

var app4 = new Vue({
  el: '#app-4',
  data: {
    todos: [
      { text: '学习 JavaScript' },
      { text: '学习 Vue' },
      { text: '整个牛项目' }
    ]
  }
})

使用 v-for 指令能够渲染出一个个的列表，能够将 todos 数组中的对象都渲染到页面上。

处理用户输入

我们可以通过 v-on 指令添加一个事件监听器，通过它调用 Vue 在实例中定义的方法：

<div id="app-5">
  <p>{{ message }}</p>
  <button v-on:click="reverseMessage">逆转消息</button>
</div>

var app5 = new Vue({
  el: '#app-5',
  data: {
    message: 'Hello Vue.js!'
  },
  methods: {
    reverseMessage: function () {
      this.message = this.message.split('').reverse().join('')
    }
  }
})

我们在 Vue 的选项中添加一个 methods 属性，里面有我们定义的方法，当 v-on 监听的事件触发了，会自动调用存储在这里的方法。

组件

组件系统是Vue的一个重点，它可以将我们要用到的片段抽出来，可以很方便的复用。

在 Vue 里，一个组件本质上是一个拥有预定义选项的一个 Vue 实例。

在Vue中注册组件很简单：

Vue.component('todo-item', {
  template: '<li>这是个待办项</li>'
})

现在我们可以用它构建另一个组件模板：

<ol>
    <todo-item></todo-item>
</ol>

但是这样子的话，每个组件的文本都是相同的。我们应该能从父作用域将数据传到子组件才行。我们来修改一下组件的定义，使得组件可以接受一个 prop （也就是属性）:

Vue.component('todo-item', {
  // todo-item 组件现在接受一个
  // "prop"，类似于一个自定义特性。
  // 这个 prop 名为 todo。
  props: ['todo'],
  template: '<li>{{ todo.text }}</li>'
})

可以看到，我们能够在模板中读取 props 属性的值。

现在，我们可以使用v-bind 指令将代办事项传到循环输的每个组件中：

<div id="app-7">
  <ol>
    <!--
      现在我们为每个 todo-item 提供 todo 对象
      todo 对象是变量，即其内容可以是动态的。
      我们也需要为每个组件提供一个“key”，稍后再
      作详细解释。
    -->
    <todo-item
      v-for="item in groceryList"
      v-bind:todo="item"
      v-bind:key="item.id"
    ></todo-item>
  </ol>
</div>

Vue.component('todo-item', {
  props: ['todo'],
  template: '<li>{{ todo.text }}</li>'
})

var app7 = new Vue({
  el: '#app-7',
  data: {
    groceryList: [
      { id: 0, text: '蔬菜' },
      { id: 1, text: '奶酪' },
      { id: 2, text: '随便其它什么人吃的东西' }
    ]
  }
})
