MySQL 基础操作

# 连接数据库
$ mysql -u root -p
Enter password: ****

-u 指定用户， -p 输入密码。

退出 MySQL 交互式命令行：

mysql> exit
Bye

使用 PHP 连接MySQL

打开连接

mysqli_connect(host,username,password,dbname,port,socket);

参数都是可选的。

关闭 mysqli 连接

mysqli_close(mysqli $link) : bool

关闭指定的连接，如果没有指定连接，则默认关闭上一个打开的连接。通常不需要使用次函数关闭连接，因为已打开的非持久化连接会在脚本执行完毕后自动关闭。

$connInfo = [
    'host' => 'localhost',
    'port' => '3306',
    'passwd' => 'root',
    'user' => 'root',
    'db' => 'test'
];

$conn = mysqli_connect("{$connInfo['host']}:{$connInfo['port']}", $connInfo['user'], $connInfo['passwd']);
if (!$conn) {
    die("error" . mysqli_errno($conn));
}
echo "connection successful.\n";
mysqli_close($conn);

创建数据库

我们可以在 mysql 交互式命令行上创建命令：

CREATE DATABASE db_name;

也可以使用 mysqladmin 创建数据库：

$ mysqladmin -u root -p create db_name
Enter password: ****

上面的命令执行结束后会创建一个名为db_name 的数据库。

+--------------------+
| Database           |
+--------------------+
| db_name            |
| information_schema |
| mysql              |
| performance_schema |
| sys                |
+--------------------+

PHP 创建数据库

php 可以使用 mysqli_query 函数创建或删除 MySQL 数据库。

语法如下：

mysqli_query(mysqli $link, string $query [, int $resultmode]): mixed
parametricdesciprtion
$link必需。规定使用的 mysql 连接。
$query必需。指定的查询字符串。
$resultmode可选。一个常量。可以是下面的任意一个：MYSQLI_USE_RESULT（检索大量数据，可以使用这个）、 MYSQLI_STORE_RESULT(默认)。
$connInfo = [
    'host' => 'localhost',
    'port' => '3306',
    'passwd' => 'root',
    'user' => 'root',
    'db' => 'phplearn'
];

$conn = mysqli_connect("{$connInfo['host']}:{$connInfo['port']}", $connInfo['user'], $connInfo['passwd']);

if (!$conn) {
    die("error" . mysqli_errno($conn));
}

echo "connection successful.\n";

$sql = "CREATE DATABASE {$connInfo['db']}";
$retval = mysqli_query($conn, $sql);

if (!$retval) {
    die("Create database failed.\n" . mysqli_error($conn));
}

echo sprintf("Create Database %s successful.\n", $connInfo['db']);

删除数据库

drop 命令

删除数据库十分简单，在登录了 mysql 交互式命令行以后使用 drop 命令就可以删除了。

DROP DATABASE db_name;

上述命令执行以后会删除一个名字叫做 db_name 的数据库。

当然也可以使用 mysqladmin 删除数据库：

mysqladmin -uroot -p drop db_name
Enter password: ****
Dropping the database is potentially a very bad thing to do.
Any data stored in the database will be destroyed.

Do you really want to drop the 'db_name' database [y/N] y
Database "db_name" dropped

使用 mysqladmin 删除数据库会有提示；确认删除数据库，输入y就可以删除了。

使用 PHP 删除数据库

删除数据库同样是使用 mysqli_query 。

$connInfo = [
    'host' => 'localhost',
    'port' => '3306',
    'passwd' => 'root',
    'user' => 'root',
    'db' => 'phplearn'
];

$conn = mysqli_connect("{$connInfo['host']}:{$connInfo['port']}", $connInfo['user'], $connInfo['passwd']);
if (!$conn) {
    die("error" . mysqli_errno($conn));
}
echo "connection successful.\n";

$sql = "DROP DATABASE {$connInfo['db']}";
mysqli_query($conn, $sql);
if (!$retval) {
    die("Drop database failed.\n" . mysqli_error($conn));
}

echo sprintf("Drop Database %s successful.\n", $connInfo['db']);

数据库操作

登录了 mysql 命令行以后，可以很轻松的选择不同的数据库来操作：

$ mysql -u root -p
Enter password: ****

mysql> show databases;
+--------------------+
| Database           |
+--------------------+
| database           |
| information_schema |
| mysql              |
| performance_schema |
| phplearn           |
| sys                |
+--------------------+
7 rows in set (0.00 sec)

可以看到有一个数据库的名字叫做 database ，这个可是 mysql 的关键字，怎么创建的呢？

mysql> create database `database`;
Query OK, 1 row affected (0.09 sec)

使用反引号就可以创建这个关键字 的数据库名啦。

mysql> drop database `database`;
Query OK, 0 rows affected (0.04 sec)

删除也要用反引号，不然就会这样：

mysql> drop database database;
ERROR 1064 (42000): You have an error in your SQL syntax; check the manual that corresponds to your MySQL server version for the right syntax to use near 'database' at line 1

选择数据库：

mysql> use phplearn;
Database changed

使用 PHP 选择数据库

php 提供了函数 mysqli_select_db 来选取一个数据库。函数在执行成功后返回 TRUE ，否则返回 FALSE 。

mysqli_select_db(connection, db_name);

上面的两个参数应该不用解释了。这个函数很简单，指定连接，指定数据库名就可以了。

$connInfo = [
    'host' => 'localhost',
    'port' => '3306',
    'passwd' => 'root',
    'user' => 'root',
    'db' => 'phplearn'
];

$conn = mysqli_connect("{$connInfo['host']}:{$connInfo['port']}", $connInfo['user'], $connInfo['passwd']);
if (!$conn) {
    die("error" . mysqli_errno($conn));
}
echo "connection successful.\n";

if (!mysqli_select_db($conn, 'phplearn')) {
    die("error: " . mysqli_error($conn));
}

mysqli_close($conn);

MySQL 创建数据表

语法：
CREATE TABLE table_name (column1_name column1_type, column2_name column2_type, ...);

下面是一个例子：

CREATE TABLE IF NOT EXISTS `runoob_tbl`(
   `id` INT UNSIGNED AUTO_INCREMENT,
   `title` VARCHAR(100) NOT NULL,
   `author` VARCHAR(40) NOT NULL,
   `date` DATE,
   PRIMARY KEY ( `id` )
)ENGINE=InnoDB DEFAULT CHARSET=utf8;
