webpack 学习

最近发现，自己对前端一无所知，所以，准备好好的学一下前端，就从 webpack 开始吧！

所以找了大神的 webpack 教程，一路学过来，学的过程中做点笔记，以防以后忘记了。

放张图，避免尴尬。

概念

在官网的文档上说了，我们需要先理解 4 个概念：

  • 入口（entry）
  • 输出（output）
  • loader
  • 插件（plugins）

入口

也就是 webpack 从哪里开始寻找依赖关系，进入入口后，webpack 会找出各个模块之间的依赖关系。

我们可以在 webpack 配置中配置 entry 属性，来指定一个或多个入口起点，默认值为 ./src。

下面是一个例子：

module.export = {
    entry: './path/to/my/entry/file.js';
}

根据应用程序的特定需求，可以以多种方式配置 entry 属性。从入口起点章节可以了解更多信息。

出口（输出）

可以在配置中配置 output 属性来告诉 webpack 在哪里输出 bundles，以及如何命名这些文件，默认值是 ./dist 。我们可以这样来配置 output 属性：

const path = require('path');

module.exports = {
    entry: './path/to/my/entry/file.js',
    output: {
        path: path.resolve(__dirname, 'dist'),
        filename: 'first-bundle-file.js'
    }
}

安装

在这之前，我们需要先安装 webpack ，但是安装 webpack 需要安装 node.js 以及 npm :

$ node --version
v8.11.3

$ npm --version
6.9.0

安装命令如下：

# 先初始化项目，使用 npm init 初始化一个项目，也就是创建 package.json 文件
$ npm init
# 可以先一路回车，最后输入 yes 就ok。

# 安装 webpack：
$ npm install webpack webpack-cli --save-dev

# 或者用简写的形式安装：
$ npm i webpack webpack-cli -D
# --save     模块会被添加到 dependencies，简化为 -S。
# --save-dev 磨矿会被添加到 devDependencies,简化为 -D。
package.json
{
  "name": "learn-webpack",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "author": "",
