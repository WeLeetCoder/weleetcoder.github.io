vscode 配置eslint自动格式化

在配置之前，需要安装 3 个插件：

  • ESLint
  • Prettier - Code formatter
  • Vetur

安装完成后，在项目目录下的 .eslintrc.js 配置为如下：

module.exports = {
  root: true,
  env: { node: true },
  extends: [
    'plugin:vue/base',
    'plugin:vue/essential',
    'plugin:vue/recommended',
    '@vue/prettier'
  ],
  rules: {
    'no-console': process.env.NODE_ENV === 'production' ? 'error' : 'off',
    'no-debugger': process.env.NODE_ENV === 'production' ? 'error' : 'off',
    'prettier/prettier': [
      'error',
      {
        singleQuote: true,
        trailingComma: 'none',
        bracketSpacing: true,
        jsxBracketSameLine: true,
        semi: false
      }
    ]
  },
  parserOptions: { parser: 'babel-eslint' },
  plugins: ['vue']
}

可以在项目目录下新建 .prettierrc :

{
  "semi": false,
  "singleQuote": true,
  "tabWidth": 2,
  "trailingComma": "none"
}
